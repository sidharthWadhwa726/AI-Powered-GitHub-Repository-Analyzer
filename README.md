# AI-Powered GitHub Repository Analyzer

An AI-powered system that analyzes GitHub repositories using static code analysis and Retrieval-Augmented Generation (RAG) to help developers quickly understand unfamiliar codebases.

## Features

- Multi-language repository analysis (8+ languages)
- Tree-sitter based static code analysis
- Folder hierarchy extraction
- Function call graph generation
- Import dependency graph generation
- Semantic code chunking
- ChromaDB vector database
- Repository-aware conversational Q&A
- Architecture summarization

## Tech Stack

- Python
- FastAPI
- Tree-sitter
- Sentence Transformers
- ChromaDB
- LangChain
- HTML/CSS/JavaScript
